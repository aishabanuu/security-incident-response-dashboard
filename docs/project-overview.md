# Project Overview

The Security Incident Response Dashboard project was developed to demonstrate how Security Information and Event Management (SIEM) solutions can be used to monitor, detect, investigate, and respond to cybersecurity threats.

The solution uses Wazuh as the SIEM platform and VMware Fusion as the virtualization environment.

The project simulates real-world security incidents using Kali Linux while Ubuntu Server acts as the centralized monitoring platform.

Objectives:

- Collect logs from endpoints
- Detect malicious activity
- Investigate alerts
- Automate incident response
- Integrate threat intelligence feeds
- Generate security reports
